API Access

The system provides RESTful API endpoints for integration with other systems.

API Base URL: /api

Authentication: JWT token required in Authorization header

Client Endpoints
  • GET /api/clients - List clients
  • GET /api/clients/{id} - Client details
  • POST /api/clients - Create client
  • PUT /api/clients/{id} - Update client
Program Endpoints
  • GET /api/programs - List programs
  • GET /api/programs/{id} - Program details
  • POST /api/programs - Create program
  • PUT /api/programs/{id} - Update program
Enrollment Endpoints
  • GET /api/clients/{id}/programs - List enrollments
  • POST /api/clients/{id}/programs/{program_id} - Enroll client
  • DELETE /api/clients/{id}/programs/{program_id} - Remove enrollment
